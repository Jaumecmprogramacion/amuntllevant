Web de Noticias Deportivas y datos de Levante UD
Este proyecto es una web diseñada para ofrecer las últimas noticias deportivas, obtenidas de los principales periódicos deportivos a través de sus feeds RSS. La web está construida con HTML, CSS y JavaScript, con un enfoque principal en la experiencia móvil, garantizando una interfaz limpia y responsiva que se adapta a diferentes tamaños de pantalla.

Características
Diseño Responsivo: Optimizado para dispositivos móviles, brindando una experiencia fluida y cómoda para los usuarios en smartphones.
Noticias Deportivas: La web obtiene información en tiempo real de los principales periódicos deportivos mediante sus feeds RSS.
Datos adicionales: Utiliza Python y diversas APIs para obtener información adicional de webs oficiales y otras fuentes relevantes. Los datos se procesan y transforman a formato JSON para ser consumidos por la web.
Tecnologías utilizadas:
Frontend: HTML, CSS, JavaScript
Backend: Python (para la obtención y procesamiento de datos)
APIs: Diferentes fuentes de datos deportivas y oficiales
Cómo Funciona
Obtención de noticias: Los feeds RSS de periódicos deportivos alimentan la web con las noticias más recientes.
Procesamiento de datos: Con Python y diferentes APIs, se obtienen y transforman otros datos relevantes (por ejemplo, estadísticas deportivas, calendarios de partidos, etc.) en formato JSON.
Visualización: La información procesada se presenta de manera clara y ordenada en la interfaz web, adaptándose a cualquier dispositivo.
